Break Time: Never forget to take your breaks again! Just set it, and forget it! With break_time!

Requirements: 
GIT
Python 2.7+ (tested)

Installation:
From the terminal, perforn the following steps:
- Clone into home directory using "git clone http://www.githib.com/TAPinson/break_time.git"

Example Usage:
- Navigate to program directory
- From the terminal run command "python break_time
- Do stuff (two hours)
- Change to to the project directory using "cd break_time"
- You will then be directed to Kool and the Gang! A break will be had, and you'll be refreshed and ready to go!

Development Setup:
Python2.7
Python Time Module
Python OS Module
